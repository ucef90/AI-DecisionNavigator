# Scoring Engine

## Objectif

Transformer les réponses du questionnaire en évaluation structurée.

---

# Axes de scoring

## Axe 1 — Clarté du besoin

Évaluer :
- maturité métier ;
- clarté ;
- objectifs ;
- utilisateurs.

---

## Axe 2 — Pertinence IA

Évaluer :
- nécessité réelle de l’IA ;
- valeur IA ;
- possibilité automatisation simple.

---

## Axe 3 — Données

Évaluer :
- disponibilité ;
- qualité ;
- accessibilité ;
- sensibilité.

---

## Axe 4 — Valeur métier

Évaluer :
- gain temps ;
- ROI ;
- amélioration qualité ;
- valeur utilisateur.

---

## Axe 5 — Faisabilité technique

Évaluer :
- intégration SI ;
- APIs ;
- architecture ;
- maintenance.

---

## Axe 6 — Risques

Évaluer :
- RGPD ;
- hallucination ;
- biais ;
- sécurité ;
- criticité métier.

---
