# Decision Engine

## Objectif

Produire une décision argumentée à partir :
- du questionnaire ;
- du scoring ;
- des workflows ;
- des risques ;
- de la gouvernance ;
- des cartographies.

---

# Décisions possibles

## GO IA

Conditions :
- besoin mature ;
- données disponibles ;
- risques maîtrisables ;
- valeur forte.

---

## POC

Conditions :
- potentiel intéressant ;
- faisabilité incertaine ;
- besoin de validation.

---

## Automatisation simple

Conditions :
- règles stables ;
- faible complexité ;
- IA non nécessaire.

---

## Étude complémentaire

Conditions :
- besoin immature ;
- manque données ;
- workflow non maîtrisé.

---

## NO GO

Conditions :
- risques élevés ;
- faible valeur ;
- contraintes réglementaires ;
- faisabilité faible.

---

# Vision future

Le moteur devra :
- générer des recommandations ;
- expliquer les décisions ;
- proposer des plans d’action ;
- détecter les blocages ;
- identifier les prérequis ;
- recommander des architectures IA.
